pkgzip

pkgzip gives you a webpack'd bundle for your npm packages.

To get started try playing with this URL: https://pkgzip.com/bundle.js?packages=left-pad

Features

  • Caches results server-side for fast 2nd-time responses
  • Grabs requested packages using yarn (much faster)
  • Supports fixed (1.9.0) and flexible (@1.x / @^1.2.0 / @latest / etc) semver ranges
  • Minification (?flags=minify)
  • Deduplication (?flags=minify,dedupe)
