pkgzip beta

Your npm packages, webpacked with dependencies.

What's the URL I should use for script tags?

<script src="https://pkgzip.com/bundle.js?packages=left-pad"></script>

You'll now have window.leftPad available to use.

Can I just use unpkg/Gomix/Webpackbin instead?

Maybe, those tools are awesome! Pkgzip is useful when you want to use packages with dependencies in your own environment, such as a static HTML file or a Codepen.

Should I use this on my production site?

Nope. It's great for prototyping but your production users will probably notice some lag.
