Frogmarch

Frogmarch is a service which gives you a simple bundle of any npm packages you ask for. It uses webpack to create the bundle.

This is handy when you want to create a simple site without adding webpack to your build pipeline.

You can get going by including a script such as:

<script src="https://frogmarch.internal.app.dev.atlassian.io/bundle.min.js?packages=ak-navigation@latest,ak-icon@8.x"></script>

API

You can use semver ranges when specifying package versions. Results are cached to S3.

Please see frogmarch/README.md for the full details.

Use it with AtlasKit

You can build your own custom bundle of AtlasKit components via the AtlasKit Registry.
